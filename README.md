# Career & Skills Hub

Career & Skills Hub is a full-stack web platform designed to combine career discovery, task management, and professional knowledge sharing in a single platform.

## Platform Modules

### 1. Job Board
- Browse and search job listings
- Filter jobs by category
- Save favorite jobs
- AI-powered job recommendations

### 2. Task & Project Management
- Create tasks and projects
- Track progress (To Do / In Progress / Done)
- Set deadlines and reminders

### 3. Knowledge Exchange
- Experts publish articles and tutorials
- Users can comment and rate content
- AI suggests articles based on user interests

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL

Authentication:
- JWT

## Features
- AI-powered recommendations
- Bilingual interface (English / Arabic)
- Responsive design
- Role-based access control (User / Admin / Expert)

## Project Goal
Build a unified platform that helps users discover career opportunities, manage productivity, and share professional knowledge in one ecosystem.
