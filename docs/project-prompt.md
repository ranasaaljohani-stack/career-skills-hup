# Project Prompt - Career & Skills Hub

## Original Prompt Used to Build This Project

Build a full-stack web platform called **Career & Skills Hub** that integrates three main modules:

### 1. Job Board
- Browse, search, and filter job listings
- Save favorite jobs
- AI-powered job suggestions based on user profile and activity
- Admin can add, edit, and remove job postings

### 2. Task & Project Management
- Create personal tasks and projects with statuses (To Do / In Progress / Done)
- Set deadlines with reminders
- Basic productivity dashboard with progress tracking

### 3. Knowledge Exchange
- Experts post articles and short tutorials
- Users can comment and rate articles
- AI suggests articles based on user interests
- Admin can approve or remove content

---

## Technical Requirements

- **Frontend:** React + Tailwind CSS — responsive, bilingual (English & Arabic)
- **Backend:** Node.js + Express (single unified API)
- **Database:** PostgreSQL (users, jobs, tasks, articles, favorites)
- **Auth:** JWT-based with three roles — User / Admin / Expert

---

## Core Features
- Unified dashboard showing saved jobs, tasks, and suggested content
- Notifications for jobs, deadlines, and new articles
- AI recommendations for jobs and articles based on user skills/activity
- English & Arabic support with auto UI direction switch (LTR/RTL)
